Meme Bot

Project status: completed

Line of codes: over 800 (4 modules + 3 extra scripts)

Description
Meme Bot is a program that downloads popular memes from various subreddits and re-post them every hour on an Instagram page.

Features include Reddit meme scrapper, profanity censor, smart tags generator and Instagram post scheduling.

Additional notes
There was not a huge demand for this kind of Instagram page (couldn't get over 200 followers), thus I decided to discontinue the project.
