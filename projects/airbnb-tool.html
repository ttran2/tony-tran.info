Airbnb Tool

Project status: on hold

Line of codes: currently approx. 1500 (9 modules)

Description
Airbnb Tool is a desktop application that helps Airbnb hosts to store guests information locally and compile reports for government institutes.

In the Czech Republic, for the sake of safety, hosts are obliged to send regular reports to various government institutes and offices. These reports include some sensitive information such as guest's passport number, thus this application will have multiple layers of security to ensure none of the sensitive data would get leaked.

Airbnb Tool features include user-friendly GUI, storing and retrieving all guests personal information from an encrypted SQLite database, generating various reports for various government institutions and offices (Police report, Statistical report, Tourist Fee report) and displaying the status of each Airbnb room.
