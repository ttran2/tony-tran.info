Liquid Management Automatic Organizer

Project status: discontinued

My Role: back end developer and sole database developer

Product Description
Liquid Management Automatic Organizer is an IT solution for a fictional transportation firm (the clients are played by our teachers). This company specializes in transportation of different fluids (some can even be hazardous). In a nutshell, this IT solution should transform the company's paper-based office to a software-based administration system. It is responsible for various tasks accross several departments, such as managing the divers, the trailers, and the trucks.

Project notes
For this project, me and my team were very ambitious. We didn't want to just simply tick all the requirements, but instead use this project as a place to try out new techniques, tools and technologies that are out of our school curriculum. We use 2 custom libraries (currently its a private Maven repository), we use latest Java 14 (experimental) features at that time (records), we have tests, we have scripts generating Java codes based on SQL script, and much more.
