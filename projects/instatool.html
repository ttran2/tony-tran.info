InstaTool

Project status: testing

Line of codes: over 1800 (6 modules)

Description
InstaTool is a desktop application currently only used by me and my friend to maintain our Instagram sites. Our Instagram sites feature Instagram content creators with great potential and therefore gives them the opportunity for more recognition, enables them to grow exponentially and interact with the international community of Photographers.

InstaTool features include downloading high-resolution images from a list of hand-selected Instagram content creators, generating Instagram captions based on a customizable template, generating a list of popular tags and scheduling Instagram posts via Hootsuite. The user selects images to be scheduled via the user-friendly GUI, which was created using the Tkinter GUI framework.

Additional notes
There are two versions of this application. The first version (v1.X) schedules posts immediately after the user selects an image. This would, however, force the user to wait for a minute after every image selection. The latest version of this application schedules all images after the user finish selecting the images.
