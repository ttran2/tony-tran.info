ValorantStats

Project status: completed

Line of codes: currently approx. 800 (14 modules)

Description
ValorantStats is a desktop application, which display to the user their Valorant game hidden statistics fetched from Valorant In-Game API. At the time of development, Valorant did not display the user's rank progress (how far till they reach next rank). This application main selling point is visually showing the users progress and exactly how far they are from next rank.

Valorant is a 5v5 FPS shooter game, developed, published, and maintained by Riot Games. It is currently, one of the most popular game in their respective genre. As of patch 2.0, Valorant had made user's rank progress visible in-game, thus making this desktop application redundant.

Additional notes
This main goal of this project was for me to learn and gain experience in developing with the cross-platform Python GUI library PyQt. This project is one of the few project, where I could showcase my front-end GUI design skills, considering most of my project has CLI (Command-Line Interface).
