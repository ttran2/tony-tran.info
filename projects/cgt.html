Csgo Global Talk (CGT)

Project status: on hold

Line of codes: over 1000 (9 modules)

Description
Csgo Global Talk (CGT) is a command-line interface (CLI) application, which lets you understand and communicate in different languages in CS:GO games, making communication between gamers across the world much easier.

CS:GO is a first-person shooter game where communication between teammates is crucial. However, since gamers come from all part of the world, the language barrier poses an issue. CGT uses Google Translate to translate all chat communication between players. In the future, this application might support text-to-speech and speech-to-text.
