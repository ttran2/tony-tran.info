CsMoney Bot

Project status: completed

Line of codes: over 1200 (3 modules)

Description
CsMoney Bot is a desktop application which analyzes CsMoney stock to Steam Market. In short, it notifies the user on decent trade deals and opportunities.

CsMoney is a 3rd party website involve in trading with in-game items. This application analyzes CsMoney item prices with the Steam Market prices and notifies the user on potentially good deals based on the user's customizable criteria. The application is also capable of categorizing deals into how good the deals are. This applications interface has a lot of user-friendly elements to maximize the user's efficiency.

Additional notes
